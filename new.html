Question-1: What are the different ways to select an element in the DOM?

We can select elements using methods like getElementById, getElementsByClassName, getElementsByTagName, querySelector, and querySelectorAll.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML returns all the HTML content inside an element, including tags. innerText returns only the visible text content, while textContent returns all text content, even if it's hidden.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where we can attach a single event listener to a parent element to handle events that occur on its child elements

Question-4: What is event bubbling in the DOM?

Event bubbling is when an event triggered on a nested element also triggers event listeners on its parent elements in the hierarchy, going up towards the root of the document.

Question-5: How do you create, add, and remove elements using JavaScript?

We can create new elements with document.createElement(), add them to the DOM with appendChild() or insertBefore(), and remove them with removeChild()
